Projects

Myrmica (Unreal 5.3 C++)

Published game on steam

Myrmica is an isometric investigation game set in a mysterious Petrol Station stuck in a time-loop. The game features a diverse cast of 5 Non-Playable Characters who will follow a set of routines every day and the player must investigate these and manipulate them in order to stop the Time-Loop.

As a programmer i developed key systems including all the UI (using Common UI as base), dynamic player camera, a specific light system for the game, wall occlusion with the player, input handling, and the visualization of the line-of-sight. Contributed additional features and optimizations across the game.

API to connect a Smart Rubiks cube with Unity

Computer science's Development final degree

In this project I developed an interface to connect a Giiker smart cube (Rubiks cube) with Unity so it could be used as an input device. This project is composed of two parts. In the first part, I developed the Bluetooth connection of the smart cube with a C# project and then the plugin to connect it to Unity. In the second part of this project, I created a prototype of a physical widget that uses Arduino with Bluetooth as its core, providing feedback to events occurring in the game I developed as a demo for the controller (it was made ad hoc). This project received a perfect grade of 10. In the following link, there is the document (in Spanish) where everything is detailed.

Simple tower defense

Unreal 5.3 UI Project

Tower defense game where the goal is to survive as many days as possible. The project uses Common UI to manage input handling across different menus, making the user interface responsive to various input devices. Although the game is short, my goal was to create a polished, well-structured experience that showcases my skills with Unreal UI and can be expanded over time. The average Shopping Experience

C++ with own engine game

The average shopping experience is a game I made using my own 2Dmotor in C++ about a little guy who goes at the supermarket to buy some things but must confront people he doesn’t want to see.
